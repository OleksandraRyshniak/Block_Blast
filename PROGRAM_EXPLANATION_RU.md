# 📱 Block Blast — Описание программы

## 🎮 Общий обзор

**Block Blast** — это .NET MAUI приложение-игра головоломка, похожая на "Tetris" или "Brick Blast". Игрок размещает цветные блоки на 8×8 сетке, и когда полная строка или столбец заполняется, они уничтожаются, приносив очки. Цель — набрать максимально возможное количество очков до конца игры.

**Платформы:** iOS, Android, Windows  
**Фреймворк:** .NET MAUI (cross-platform UI)  
**Язык:** C#

## 🎯 Основные компоненты и их функции

### 1️⃣ **Game.cs** — Главный контроллер игры

**Назначение:** Управляет логикой игры, состоянием, событиями.

**Ключевые свойства:**
```csharp
public Board Board              // 8×8 сетка
public Player Player            // Текущий игрок
public List<Block> NextBlocks   // 3 следующих блока
public bool IsRunning          // Игра в процессе?
public bool IsOver             // Игра завершена?
public GameMode _mode          // Easy или Hard
```

**Ключевые методы:**

| Метод | Описание |
|-------|---------|
| `Start()` | Начинает новую игру. Инициализирует доску, первые 3 блока. В режиме Hard заполняет доску на 12 клеток. |
| `Stop()` | Останавливает игру (пауза). |
| `TryPlaceBlock(blockIndex, row, col)` | Попытается разместить блок на доске. Возвращает `true` если успешно. |
| `OnLinesCleared(count, lines, isBoardClear)` | Private обработчик — когда линии уничтожены, обновляет очки игрока. |

**События (delegates):**
```csharp
GameStarted         // Игра началась
GameOver            // Игра окончена
ScoreChanged(int)   // Очки изменились
LinesCleared(...)   // Линии уничтожены
BoardUpdated        // Доска обновилась
```

### 2️⃣ **Board.cs** — Игровая доска

**Назначение:** Представляет 8×8 сетку, проверяет правильность ходов, уничтожает строки/столбцы.

**Константы:**
```csharp
public const int Rows = 8;  // Высота
public const int Cols = 8;  // Ширина
```

**Ключевые методы:**

| Метод | Описание |
|-------|---------|
| `CanPlace(block, row, col)` | Проверяет, может ли блок быть размещён в данной позиции (не выходит за границы, нет пересечений). |
| `PlaceBlock(block, row, col)` | Размещает блок на доске, заполняет ячейки, затем вызывает `ClearFullLines()`. |
| `ClearFullLines()` | Private — ищет и уничтожает полные строки и столбцы. |
| `IsRowFull(row)` / `IsColFull(col)` | Проверяет, полна ли строка/столбец. |
| `ClearRow(row)` / `ClearCol(col)` | Очищает строку/столбец. |
| `IsBoardEmpty()` | True если доска полностью пуста. |
| `CanPlaceAnywhere(block)` | Проверяет, может ли блок быть размещён ВГДЕ-НИБУДЬ на доске. |
| `IsGameOver(nextBlocks)` | True если ни один из 3 следующих блоков не может быть размещён. |
| `GetPreviewCells(block, row, col)` | Возвращает список клеток, где будет блок (для preview UI). |
| `GetWouldClearLines(block, row, col)` | Возвращает, какие строки/столбцы будут уничтожены **если** разместить блок. |
| `FillRandom(count)` | Заполняет `count` случайных клеток (для режима Hard). |
| `Reset()` | Очищает доску. |

**Статические фигуры (factory методы):**

### 5️⃣ **Player.cs** — Данные игрока

**Назначение:** Хранит имя, очки, рекорд, комбо, статистику.

**Свойства:**
```csharp
public string Name              // Имя игрока
public int Score                // Текущие очки
public int BestScore            // Рекордные очки
public int ComboCount           // Текущее комбо (сколько подряд линий)
public int MaxCombo             // Максимальное комбо за игру
public int TotalLinesCleared    // Всего строк/столбцов уничтожено
```

**Методы:**

| Метод | Описание |
|-------|---------|
| `AddPlacementScore(cellsPlaced)` | Добавляет очки за размещение (кол-во клеток × 1). |
| `AddLineScore(linesCleared, isBoardClear)` | Добавляет очки за уничтожение линий. Увеличивает комбо. |
| `ResetCombo()` | Сбрасывает текущее комбо. |
| `ResetScore()` | Сбрасывает всю статистику текущей игры. |

**Система очков:**

1. **За размещение блока:** `количество_клеток × 1`
2. **За уничтожение линий:** базовый бонус зависит от количества:
   - 1 линия → 10 очков
   - 2 линии → 20 очков
   - 3 линии → 60 очков
   - 4 линии → 120 очков
   - 5+ линий → 300 очков

3. **Комбо-мультипликатор:** каждое следующее уничтожение линий подряд даёт `1.0 + (ComboCount - 1) × 0.5` множитель
   - 1-е уничтожение: ×1.0
   - 2-е уничтожение: ×1.5
   - 3-е уничтожение: ×2.0
   - Итд.

4. **Очистка доски:** +360 бонус если уничтожена вся доска

## 🎮 Поток игры

### Этап 1: Запуск
```
1. Пользователь запускает приложение
2. Проверяется логин (AccountService)
3. Если не залогинен → LoginPage
4. Если залогинен → StartPage (главное меню)
```

### Этап 2: Начало игры
```
1. Пользователь нажимает "Play Easy" или "Play Hard"
2. Создаётся объект Game с режимом
3. Вызывается Game.Start()
4. В режиме Hard — доска заполняется на 12 случайных клеток
5. Генерируются 3 следующих блока (BlockFactory)
6. GamePage отображается
```

### Этап 3: Игровой цикл
```
1. Игрок видит текущий блок и 3 следующих
2. Игрок перетаскивает блок на доску
3. TryPlaceBlock() проверяет валидность
4. Если валидно:
   a. Блок размещается на доске
   b. Проверяются полные строки/столбцы
   c. Полные линии уничтожаются (ClearFullLines)
   d. Очки пересчитываются (с комбо-мультипликатором)
   e. Новые 3 блока генерируются
5. UI обновляется (BoardUpdated event)
6. Если ни один из 3 блоков не может быть размещён → Game Over
```

### Этап 4: Завершение
```
1. IsGameOver() вернул true
2. GameOver event → OnGameOver() вызывается
3. Результат сохраняется (ScoreService.Save)
4. Показывается экран окончания с рейтингом
5. Пользователь может вернуться в меню или начать заново
```

## 🎯 Основной цикл алгоритма

```csharp
while (IsRunning && !IsOver)
{
    // 1. Показать 3 следующих блока
    DisplayNextBlocks();

    // 2. Ждать выбора игрока
    WaitForPlayerInput();

    // 3. Попытаться разместить блок
    if (TryPlaceBlock(selectedBlockIndex, selectedRow, selectedCol))
    {
        // 4. Проверить и уничтожить полные линии
        ClearFullLines();

        // 5. Обновить очки с комбо
        UpdateScore();

        // 6. Сгенерировать новые 3 блока
        NextBlocks = GetNext3();

        // 7. Обновить UI
        RefreshBoard();
    }

    // 8. Проверить, может ли ещё что-то быть размещено
    if (IsGameOver(NextBlocks))
    {
        IsOver = true;
        SaveScore();
        ShowGameOverScreen();
    }
}
```
